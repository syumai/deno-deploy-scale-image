# deno-deploy-scale-image

image scaling app built on Deno Deploy and Go WebAssembly.

## Usage

### Run app

```
deployctl run --no-check mod.js
```

### Scale image

show scaled image on browser

`http://0.0.0.0:8080/image?path=landscape.jpg&width=200`

available images

* syumai.png
* landscape.jpg
* ramen.jpg
